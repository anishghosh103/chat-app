Events Documentation
Events to listen:
typing event
This event ("typing") has to be listened on the user's end to know that some user is typing in the opened room.
Output Data
{
"roomId": 'string',
"userId": 'string'
}
deactivated event
This event ("deactivated") has to be listened on the user's end to know that some room has been deactivated by it's creator.
Output Data
{
"roomId": 'string'
}
activated event
This event ("activated") has to be listened on the user's end to know that some room has been activated by it's creator.
Output Data
{
"roomId": 'string'
}
created event
This event ("created") has to be listened on the user's end to know that a new room has been created.
Output Data
{
"roomId": 'string',
"name": 'string',
"description": 'string',
"creatorId": 'string',
"createdAt": Date,
"users": [{
"userId": 'string'
"lastViewed": Date
}],
"lastUpdated": Date,
"active": Boolean
}
joined event
This event ("joined") has to be listened on the user's end to know that a user has joined a room.
Output Data
{
"roomId": 'string',
"userId": 'string',
"userName": 'string',
}
left event
This event ("left") has to be listened on the user's end to know that a user has left a room.
Output Data
{
"roomId": 'string',
"userId": 'string',
"userName": 'string',
}
deleted event
This event ("deleted") has to be listened on the user's end to know that a room has been deleted by it's cretor.
Output Data
{
"roomId": 'string',
}
update event
This event ("update") has to be listened on the user's end to know that a user has send a message.
Output Data
{
"roomId": 'string',
"type": 'string', // 'chat' for message
"senderId": 'string',
"senderName": 'string',
"message": 'string',
"time": Date,
}
Events to emit:
typing event
This event ("typing") has to be emitted by the user's end to let other user know that the user is typing.
Output Data
{
"roomId": 'string',
"userId": 'string'
}
